Tips：

  1. 把出现的错误代码直接复制到百度就可找到解决方法

  2. 在下载国外的一些工具时,可以考虑下载镜像,速度飞起

简单使用

创建新文件：

hexo n "新的文章名"

生成页面：

hexo g

在本地浏览器查看效果:

hexo s

部署到远端服务器:

hexo d
